Para editar texto, simplemente introduce los caracteres en el teclado virtual como lo harías con un teclado normal. Usa la cruzeta para trasladar el cursor donde quieras. El botón spl intercambia entre el teclado normal QWERTY y los carácteres especiales. Esto es util para añadir los simbolos autoctonos españoles como las tildes o la ñ o para meter caras divertidas en tus textos como ò_ó o Ö_ö.


Tambien se incluye un motor muy sencillo de reconocimiento de escritura como una alternativa a escribir el texto, pero es bastante rudimentario. Para usarlo, solo comienza el simbolo en cualquier superficie de la pantalla inferior que esté en blanco, y si DSOrganize reconoce el caracter, este será insertado.
