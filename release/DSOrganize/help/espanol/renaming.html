Para crear o renombrar un archivo, simplemente escribe el nuevo nombre que desees para el archivo. Por favor, observa que en esta ocasión, no se comprobará los archivos creados desde 0, ten cuidado con lo que pones.

Para editar texto, simplemente introduce los caracteres en el teclado virtual como lo harías con un teclado normal. Usa la cruzeta para trasladar el cursor donde quieras. El botón spl intercambia entre el teclado normal QWERTY y los carácteres especiales.

Tambien se incluye un motor muy sencillo de reconocimiento de escritura como una alternativa a escribir el texto, pero es bastante rudimentario. Para usarlo, solo comienza el simbolo en cualquier superficie de la pantalla inferior que esté en blanco, y si DSOrganize reconoce el caracter, este será insertado.
