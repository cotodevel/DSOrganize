Para editar el titulo, inserta los caracteres usando el teclado. mientras el campo del titulo esté seleccionado, los botones &up; y &down; cambian la prioridad. usa &left; o &right; para mover el cursor.

Para editar el texto en el campo del mensaje de cada tarea, simplemente introduce los caracteres con el teclado virtual igual que lo harías con un teclado normal. Usa la cruzeta para mover el cursor. El botón spl intercambia el teclado normal con el de caracteres especiales. Esto es util si quieres añadir caracteres no ingleses. Para saltar entre los campos, usa &l; y &r; o presiona el Intro. El botón clear limpia todos los campos y el botón delete borra la entrada entera de la tarjeta.

Para usar el reconocimiento de escritura,comienza a escribir en cualquier espacio de la pantalla inferior que se encuentre en blanco, y si DSOrganize reconoce el caracter, este será escrito.
