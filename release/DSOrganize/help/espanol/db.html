La Base de datos proporciona un metodo de descargar nuevo homebrew directamente a tu DS, sin usar lector de tarjetas o PC. Actualmente la selección es pequeña, pero yo estoy intentando incrementarla diariamente.






En la pantalla principal, elige &a; para descargar o &b; para actualizar la lista. Presionando &start; te devolverá a la patalla de inicio. Puedes cancelar la descarga en cualquier momento presionando &b; durante el proceso de descarga.
