La lista TODO pretende ser un metodo facil de organizar tús tareas pendientes. Las tareas son organizados primero por prioridad, ordenandolas desde las tareas completadas, hasta las urgentes y las normales, bajo las tareas que estan siendo pospuestas.





Para seleccionar una tarea, simplemente usa &up; y &down; o tu stylus y la barra de desplazamiento.
