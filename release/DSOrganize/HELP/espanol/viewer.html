El visor de texto y html es una forma facil de ver archivos de texto y html con soporte de Marcadores. Las teclas de navegación son las mismas que en el editor de texto: &up; y &down; para subir o bajar una linea, o &l; y &r; para subir o bajar una pagina entera. Presionando &a; inserta un marcador en la posición actual. Presionando &left; o &right; saltará al siquiente o anterior Marcador, respectivamente. Presionando &b; retorna al Explorador.
