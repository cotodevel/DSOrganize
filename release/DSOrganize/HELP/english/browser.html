The file browser is provided as a simple way to manage files on your DS. To cycle through available modes, tap the &l; or &r; buttons. When viewing hidden files is enabled, pressing &x; will hide or unhide the selected file. On applicable files, pressing &y; will perform the alternate action (viewable on the top screen).

The text editor currently supports .txt, .html and .ini file editing, with syntax hilighting on .ini and .html files.
The image viewer currently supports .bmp, .jpg, .gif, .png, and some .bin formats for raw NDS data viewing.

The sound player currently supports .wav, .mp3, .ogg, .aac, .m4a, .flac, .sid, .nsf, .spc, .sndh, .pls, .m3u, and module files.

Naitive file launching is only supported on the GBAMP and SuperCard, although the MightyMax and Chishm launchers both work on most other cards.
