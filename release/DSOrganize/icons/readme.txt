DSOrganize icon sets are a simple way to customize the look of your home screen and browser.  To create a set, simply make a subdirectory here and place the image files you want under the subdirectory, and on the next boot, DSOrganize will pick it up and allow you to select it under the Configuration screen.  Extensions do not matter, DSOrganize will pick up any gif, jpg, bmp or png image.  For a list of filenames to use for your iconset, see below.  Please keep to the dimensions listed as any other dimensions on a graphic file will cause DSOrganize to not load it.  To allow transparency in your icon sets, use the color purple (255,0,255) or set the transparency color in your editor.
